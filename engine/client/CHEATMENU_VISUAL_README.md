# Визуальное меню читов - Инструкция

## Как открыть меню

### На ПК:
```
cheatmenu
```
или привяжите клавишу:
```
bind F1 "cheatmenu"
```

### Управление:
- **Мышь**: Кликните по элементам меню для взаимодействия
- **Клавиатура**:
  - `ESC` - закрыть меню / вернуться в главное меню
  - `Enter` - выбрать пункт / применить действие
  - `↑/↓` - навигация по пунктам меню
  - `Колесо мыши` - прокрутка

### На мобильных устройствах:
- Тап по элементам меню для взаимодействия
- Меню автоматически показывает курсор при открытии

## Структура меню

### Главное меню:
1. **ESP Menu** - настройки ESP
2. **Aimbot Menu** - настройки аимбота
3. **Movement Menu** - настройки движения
4. **Misc Menu** - прочие настройки
5. **Close Menu** - закрыть меню

### ESP Menu:
- ESP Enabled (ON/OFF)
- ESP Boxes (ON/OFF)
- Player Names (ON/OFF)
- Weapon Names (ON/OFF)

### Aimbot Menu:
- Aimbot Enabled (ON/OFF)
- FOV (ползунок 0-180)
- Smooth (ползунок 0.0-1.0)
- Perfect Silent (ON/OFF)
- Anti-Aim (ON/OFF)

### Movement Menu:
- Speed Multiplier (ползунок 1.0-10.0)
- Auto Strafe (ON/OFF)
- Ground Strafe (ON/OFF)
- Fake Lag (ползунок 0-500 ms)

### Misc Menu:
- Viewmodel Glow (ON/OFF)
- Block Server Commands (ON/OFF)

## Отладка

Если меню не работает:

1. Проверьте, что меню открывается:
   ```
   cheatmenu
   ```

2. Убедитесь, что курсор мыши виден при открытом меню

3. Проверьте консоль на наличие ошибок

4. Попробуйте использовать клавиатуру для навигации

## Технические детали

- Файл: `engine/client/cheatmenu_visual.c`
- Рисование: вызывается из `V_PostRender()` в `cl_view.c`
- Обработка ввода: интегрирована в `Key_Event()` и `IN_MouseEvent()`
- Поддержка тач: интегрирована в `IN_TouchEvent()`

## Известные ограничения

- Слайдеры пока не интерактивны (только отображение)
- Цветовые настройки нужно менять через консоль
- Некоторые настройки доступны только через консольные команды
